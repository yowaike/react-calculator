# React-приложение "Калькулятор"

## Описание приложения

Калькулятор - одностраничное React-приложение для выполнения арифметических операций. Приложение демонстрирует основные концепции React: работа с компонентами, состоянием, событиями и управлением пользовательским интерфейсом.

## Основные возможности

- Базовые операции: Сложение, вычитание, умножение, деление
- Ввод данных: Поддержка целых чисел и десятичных дробей
- Управление вычислениями: Очистка ввода, последовательные операции
- Интерактивный интерфейс: Мгновенное отображение результатов

## Технические особенности реализации

### Архитектура приложения
Приложение построено на функциональных компонентах с использованием React Hooks:

- useState для управления состоянием компонента
- Обработчики событий для пользовательского ввода

### Компонентная структура
App (корневой компонент)
└── Calculator (основной компонент приложения)
    ├── Display (область отображения)
    └── Keypad (клавиатура калькулятора)
        ├── NumberButtons (цифровые кнопки 0-9)
        ├── OperatorButtons (кнопки операций +, -, ×, /)
        ├── ControlButtons (кнопка очистки C)
        └── UtilityButtons (кнопки ., =)
